[convbrno.herokuapp.com](https://convbrno.herokuapp.com)

**Transport-layer:**
* Graphql

**Backend:**
* Nodejs - express
* apollo-server-express


**Frontend:**
* React 16
* Apollo Client
* Mobx
* Ant UI

**DB:**
* mongodb - Service on [mLab.com](https://mlab.com/)
